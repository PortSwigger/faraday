This extension will allow you to import requests and issues from your Burp workspace into Faraday.

Usage

  • Navigate to the new "Faraday" tab.
  • Log in to your Faraday instance.
    • Complete the Faraday Server URL with the address where your server is running. If it is the same machine, you can leave it as is.
    • Once you are sure the URL is correct, click the "Connect" button so the extension can validate your Faraday server.
    • Once connected, fill in the username and password fields, then click "Login". If a 2FA token is requested by the server, you will be asked to provide it.
  • Once authenticated, you will be able to:
    • Import the vulnerabilities you've found so far.
    • Choose if vulnerabilities should be imported automatically as soon as they are found or not (it's disabled by default).
    • Restore the extension configuration.
  • Additional functionality: Use the context menu to send issues and requests directly to Faraday.
